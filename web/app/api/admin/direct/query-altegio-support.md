# Запит до AI агента техпідтримки Altegio

**Питання:**

Привіт! Я розробляю інтеграцію з Altegio API і мені потрібно отримати для багатьох клієнтів (100-300) такі дані:
- `spent` (загальна сума витрат клієнта)
- `visits` (кількість візитів)

Зараз я використовую endpoint `GET /v1/client/{location_id}/{id}` для кожного клієнта окремо, але це дуже повільно для масового запиту.

**Питання:**
1. Чи є endpoint для масового отримання даних клієнтів з полями `spent` та `visits`?
2. Якщо так, який саме endpoint і які параметри потрібні?
3. Чи можна використовувати `POST /company/{id}/clients/search` для отримання цих полів? Які поля потрібно вказати в `fields`?
4. Чи є обмеження на кількість клієнтів в одному запиті?

**Мій поточний код:**
```typescript
// Отримую одного клієнта
const client = await getClient(companyId, clientId);
// client.spent та client.visits доступні

// Але для 300 клієнтів це занадто повільно
```

**Що мені потрібно:**
- Endpoint для масового отримання `spent` та `visits` для багатьох клієнтів одночасно
- Або спосіб оптимізувати поточний підхід
